# Парсер свежих статей Хабра

Скрипт парсит страницу свежих статей Хабра и выводит статьи, в которых найдено хотя бы одно ключевое слово.

Ключевые слова находятся в отдельном файле `keywords.txt`.


# Как изменить ключевые слова

Откройте файл `keywords.txt` и укажите ключевые слова по одному на строку.

# Парсер свежих статей Хабра

Скрипт парсит страницу свежих статей Хабра и выводит статьи, в которых найдено хотя бы одно ключевое слово.

Ключевые слова задаются в начале файла `main.py` в переменной `KEYWORDS`:

```python
KEYWORDS = ["дизайн", "фото", "web", "python"]
```

## Запуск

Установите зависимости:

```bash
pip install -r requirements.txt
```

Запустите скрипт:

```bash
python main.py
```

## Формат вывода

В стандартный вывод попадают только найденные статьи:

```text
дата – заголовок – ссылка
```

Служебные сообщения в stdout не выводятся, чтобы не нарушать формат результата.

## Логика поиска

1. Скрипт получает свежую ленту Habr.
2. Для каждой карточки отдельно собираются заголовок и preview.
3. Заголовок и preview объединяются в один текстовый блок и проверяются по списку `KEYWORDS`.
4. Если ключевые слова не найдены в карточке, дополнительно проверяется полный текст статьи.
